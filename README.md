# AI + 运维管理平台
> 基于 FastAPI + Umi.js 构建的新一代智能运维管理平台，深度融合 腾讯混元大模型，实现 AI 驱动的一键式自动化运维。
- 服务端基于Python的Fastapi框架
- Web端基于React + Umi框架
- AI 驱动智能化运维

✨ 项目亮点

    🤖 AI 驱动的智能运维：集成腾讯混元大模型，通过自然语言交互完成复杂运维任务

    ⚡ 一键式自动化：从故障诊断到修复建议，全程 AI 辅助，大幅降低人工干预

    🎯 全栈现代化技术栈：FastAPI（异步高性能后端）+ React 18 / Umi.js（企业级前端）

    📊 实时可观测性：智能日志分析、指标监控、告警预测一体化

## 🛠 技术架构
层级	技术选型
后端服务	Python 3.10+ / FastAPI / Uvicorn
AI 引擎	腾讯混元大模型（自然语言理解 + 智能决策）
前端框架	React 18 + Umi.js 4
UI 组件库	Ant Design 5.x
状态管理	Zustand
网络请求	Fetch API

## 📦 安装依赖
```
pip install -r requirements.txt
```

## 启动服务端
```
uvicorn main:app --host 0.0.0.0 --port 8000
```

## 启动Web端
```
cd /web
npm install
npm start
```

## 访问Web端
http://localhost:8001

## 🎯 核心功能模块
1. 🤖 AI 智能助手（混元大模型驱动）

    自然语言查询系统状态

    智能分析日志异常

    自动生成故障修复脚本

    运维知识库问答

2. 📊 监控大屏

    实时系统指标（CPU/内存/磁盘/网络）

    服务健康状态全景图

    智能阈值预警

3. 🔔 告警中心

    AI 驱动的告警降噪与聚合

    根因智能分析

    自动化告警处置建议

4. 📝 日志管理

    海量日志实时检索

    AI 日志语义分析

    异常模式智能识别

5. 🚀 自动化任务

    一键运维脚本执行

    AI 生成任务编排

    定时任务智能调度

    RBAC 角色权限控制

    操作审计日志
