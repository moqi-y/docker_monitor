# 容器管理系统Web端
基于React + Umi框架开发

## 项目运行

```bash
# 克隆项目
git clone 

# 进入项目目录
cd /web/ui

# 安装依赖
npm install

# 启动服务
npm start
```

浏览器访问 http://localhost:8000

## 构建生产环境

```bash
npm run build
```

构建完成后，在`/web/ui/dist`目录下生成静态文件。

## 其它

```bash
# 预览发布环境效果
npm run preview

# 代码格式检查
npm run lint

# 代码格式检查并自动修复
npm run lint:fix
```
